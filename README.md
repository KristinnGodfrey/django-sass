# Django sass
### Made with Django

### Setup:

Activate and venv:
	python3 -m venv /Users/kristinngodfrey/Environments/kristlist
	source !$/bin/activate
	
Run server:
	python3 ~/Documents/Django/kristlist/mysite/manage.py runserver

Deactivate venv:
	Deactivate
	
Deactivate server:
	sudo lsof -i tcp:8000
	kill -9 {PID}
